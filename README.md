# Mi página web personal

En este proyecto personal, crearé mi propia página web de cara a mostrar tanto mis habilidades de desarrollo web mediante la creación de la 
mencionada página, cómo otras habilidades como el análisis de datos mediante la incorporación de un portafolio con proyectos y un CV. Para 
añadir más dinamismo también crearé un blog en el que iré añadiendo mis reflexiones e ideas sobre economía, disciplina que estudié en la 
universidad. Para ello usaré principalmente los lenguajes HTML5 y CSS para el esqueleto y el estilo de la página web. A su vez, usaré Javscript
para dotar mi página web de dinamismo cuando así lo requiera.
